# Summary

Clear all Instagram media, followers, followings etc.

# Dependency

[mgp25 Instagram Private API](https://github.com/mgp25/Instagram-API)

# Install

```php
composer install
```

# Execute task

```php
cd src
php delete_media.php
php remove_followers.php
php remove_followings.php
```
